# Página Web Personal

Proyecto básico en HTML y CSS para una página web personal.

## Cómo usar

1. Cambia los datos personales en `index.html`.
2. Reemplaza `assets/foto.svg` por tu foto personal.
3. Si usas una foto JPG o PNG, cambia esta línea:

```html
<img src="assets/foto.svg" alt="Foto personal" class="photo" />
```

por algo como:

```html
<img src="assets/foto.jpg" alt="Foto personal" class="photo" />
```

4. Cambia los links de redes sociales.
5. Cambia el video de YouTube por el video que prefieras.

## Subir a GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube `index.html`, `styles.css` y la carpeta `assets`.
3. Entra a Settings > Pages.
4. En Branch selecciona `main` y `/root`.
5. Guarda y espera unos minutos.
